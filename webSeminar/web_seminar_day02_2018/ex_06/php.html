PHP

PHP, est un langage de programmation libre, principalement utilisé pour produire des pages Web dynamiques via un serveur HTTP, mais pouvant également fonctionner comme n'importe quel langage interprété de façon locale. PHP est un langage impératif orienté objet.

Remarques

  1. PHP, langage ancien, et encore très utilisé, mis à jour en 2015 avec sa version 7.
  2. PHP étant le langage recommandé souvent pour les débutants, il est souvent associé à de la mauvaise programmation.

Exemples

Exemple : Hello World.
<?php echo '<h2>Hello World</h2>'; ?>

Le code PHP peut être inclus dans une page HTML :
<!DOCTYPE html><html><body><main><article><p><?php echo 'Hello World ! ' ?></p></article><main></body><html>
