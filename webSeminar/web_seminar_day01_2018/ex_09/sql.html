SQL

SQL est un langage informatique normalisé servant à exploiter des bases de données relationnelles. La partie langage de manipulation des données de SQL permet de rechercher, d'ajouter, de modifier ou de supprimer des données dans les bases de données relationnelles.

Remarques

  1. SQL est un langage structuré facile de compréhension (car proche de phrases en anglais) mais dont les requêtes complexes demandent en revanche un jeu mental de compréhension.
  2. SQL est utilisé pour de la manipulation de données dans des bases de données de différentes tailles, même si on lui préfèrera COBOL (par exemple) pour des volumes très importants de données.

Exemples

Exemple de requête avec une table client, avec champs id, prenom, nom, ville dont on souhaite les Dupond de Strasbourg:

SELECT * FROM client WHERE ville = 'Strasbourg"
